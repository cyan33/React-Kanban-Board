## Dependencies

* React & React-DOM
* Webpack & webpack-dev-server
* Babel Core
* Babel Loader (With "es2015" and "react" presets)
* marked

## Usage

To run it locally, fork this repo and run:

`
npm install
`

then, 

`
npm start
`

then you can see the web page in http://localhost:8080 (if the port8080 is not taken)
